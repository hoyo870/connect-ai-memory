# 트레이딩 시그널 시스템 설계

## 📌 한 줄 통찰
> 지표 계산 → 조건 검사 → 시그널 생성 → 텔레그램 알림 → 주문 실행 의 5단계 파이프라인

## 시스템 구조
```
거래소 API (WebSocket)
    ↓ 실시간 캔들 데이터
지표 계산 엔진 (RSI, MACD, BB 등)
    ↓
조건 검사 (진입 조건 만족 여부)
    ↓
시그널 생성
    ↓
텔레그램 알림 (영숙)
    ↓
주문 실행 (선택: 자동 / 수동 확인)
```

## 기술 스택
- 언어: Python
- 거래소: Binance / Upbit API
- 알림: Telegram Bot API
- 스케줄링: APScheduler 또는 Celery를 이용한 주기적 작업 관리
- 데이터 저장: PostgreSQL 또는 TimescaleDB를 활용한 시계열 데이터 관리
