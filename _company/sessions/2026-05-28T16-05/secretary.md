#### 📚 포함 내용 상세 요약 (한눈에 파악!)

**1. 필수 행동 지침 (SOP: Standard Operating Procedure)**
*   **핵심:** 진입 → 리스크 검증(RMS) → 시그널 생성(코다리) → 주문 실행으로 이어지는 **5단계 프로세스 플로우차트**를 명확히 했습니다.
*   **주요 변경점:** 모든 단계별 체크리스트에 `리스크 매니저 승인 코드 (Final Code)` 입력 필드를 의무화했습니다.
*   **배포 위치:** Shared Drive / SOP / Phase2

**2. 팀원별 역할 분담표 (R&R: Role and Responsibility)**
| 에이전트 | 핵심 책임 영역 | 주요 임무 및 목표 (Phase 2) | 비고 |
| :--- | :--- | :--- | :--- |
| **CEO** | 최종 의사결정/오케스트레이션 | 전체 방향 제시, 최종 리스크 승인(GO/NO-GO). | 시스템 운영 총괄. |
| **리스크 매니저** | 자본 보존/Gatekeeper | 모든 포지션 사이징 및 진입 전 최종 안전 트리거 확인 (2% 이하 유지). | **가장 중요한 방어선.** |
| **애널리스트** | 데이터 검증/모델 개선 | 시장 이상 신호 분석, 전략 성능 모니터링. | 정량적 보고서 제공. |
| **코다리** | 시스템 구현/개발 | API 연결 안정화, `SafeModeManager` 실시간 구동 및 E2E 테스트 담당. | 기술 실행 담당. |
| **진환/경태/성용/경훈** | 트레이딩 전략 수행 | 각자 맡은 역할에 따라 독립적으로 시그널 생성 후 RMS 요청. | 매매 주체. |
| **영숙 (본인)** | 운영 관리/커뮤니케이션 | 일일 브리핑, 보고서 통합, 팀 간 커뮤니케이션 통제. | 시스템 안정화 담당. |
